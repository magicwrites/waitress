About

Waitress is a simple publishing tool, which gives you control over exposing your websites and tools to the public use. Once you will create a repository on github, you can assign it to waitress in order to automatically deploy the latest master branch. Once you are satisfied with the way it presents itself, you can deploy it further into the public environment, which can be assigned easy to access web domains.

Considerations

Waitress uses the ports from the range 3000 ‐ 9000, so when you are about to create an application which uses ports during it's runtime, please ensure those do not collide with that range.
