Roadmap

The waitress follows the semantic versioning.

Version 0.3.0

  • Disable nginx hosting when publishing website, enable afterwards. Display maintanance screen in between.
  • Utility script that will assign a domain to the waitress herself.
  • Improved quality of loadings and user interaction.
  • Improved design.

Version 0.2.0

  • Fix the breaking localStorage whenever you delete user.json file.
  • Improved quality of the websockets communication.
  • Improved socket usage to work through websocket service only.
  • Friendly panel that will allow for website domain assignment.
  • Utility script that will assign a domain to the public version of the website.
  • Utility script that will revert the public version of the website.
  • Utility script that will deploy the public version of the website.
  • Utility script that will automatically deploy the latest version of the website.
  • Replace the ubuntu with fedora.
  • Improve the configurability of the application by removing the hardcoded values.
  • Improve the interface of the application by isolating websockets.
  • Improve the stability of the development with tests.
  • Utility script that will read version data for websites.
  • Utility script that will expose public and latest version of the website on specific ports.
  • Utility script that will list the websites present in the system.
  • Utility script that will perform the initial setup of a new website.
  • Security, restriction of unauthorized access.
  • Friendly panel that will present tools to operate on website.
