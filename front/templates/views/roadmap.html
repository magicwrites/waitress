Roadmap

The waitress follows the semantic versioning convention.

Version 0.2.0

  • Utility script that will deploy the live version of the applications.
  • Utility script that will deploy the development version of the applications.

Version 0.0.2

  • Startup script that will ensure waitress services work even after host rebooting.
    • Forever service that will keep the application running even if it crashes.
    • System job that will run the forever service whenever system starts.

Version 0.1.0

  • Fix the arrows disapearance next to the header navigation while mouse pointing.
  • Utility script that will install and expose selenium service.
  • Utility script that will install nginx service.
  • Startup script that will install waitress and display this panel.
  • Friendly panel that will beautify and simplify remote server administration.

Unassigned

  • Make the waitress logo go fancy CSS rotate when you point your mouse at it.
